# Proximal-Policy-Optimization
Built a reinforcement learning system using PPO to control temperature in a simulated batch reactor. The agent learns optimal policies to maintain stability under varying conditions, demonstrating convergence through reward optimization and policy updates.

python libraries used:
numpy
pandas
torch
mathplotlib
